WP Nokia Auth

WP Nokia Auth is a WordPress plugin that allows you to manage your registered Nokia Location API credentials in a single place and, via a supplied PHP helper class, easily use them across all themes and plugins that use Nokia Location APIs on a WordPress site.

Authors

WP Nokia Auth is written, produced and hacked by Gary Gale.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/vicchi/wp-nokia-auth

WordPress Installation

You can install this plugin from the Plugins Dashboard on a self hosted Wordpress.org install or on WordPress.com, or from the official WordPress plugins repository.

Get the source code on GitHub : vicchi/wp-nokia-auth
